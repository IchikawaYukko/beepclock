４ インストール------------------------------------------------------
	アーカイブを解凍して、適当なフォルダにぶっこんで下さい。
	このファイルが読めるということは、もう解凍できていますね？

５ アンインストール------------------------------------------------------
	解凍した時にできたファイルを全て消してください。
	レジストリはいじっていません。

６ 使用法------------------------------------------------------
	speedplay3.exe をダブルクリックして起動してください。
	実行ファイル単体で動作します。

	毎時00分と30分に、マザーボード上のブザーを使って、時報を鳴らします。
	なお,EeePCのように、ブザーがついていないPCは、何も音が鳴りません(笑)
